👉Use a font-family: plus jakarta sans (This is google font and we can import it using these links in our css file)
🔗Link for fonts

@import url("https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&display=swap");
🔴To use these fonts, simply include these import statements in your style file, then set font-family: plus jakarta sans
